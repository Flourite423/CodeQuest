# 用户手册

## 前端学习APP（CodeQuest）

**版本**：V1.0  
**日期**：2026-05-30

---

## 1. 引言

### 1.1 用途

本文档面向前端学习APP（CodeQuest）的最终用户，指导学习者和管理员正确使用本系统的各项功能。

### 1.2 适用范围

- 学习者：通过移动端学习前端编程的初学者
- 管理员：负责内容运营和系统管理的运营人员

### 1.3 运行环境

**学习者端**：iOS 13+ / Android 8.0+ 智能手机  
**管理后台**：Chrome/Firefox/Edge 最新版本，桌面电脑  
**网络**：支持 WiFi/4G/5G

---

## 2. 系统简介

CodeQuest 是一款面向编程初学者的学习应用，通过"学习 → 练习 → 挑战"的游戏化闭环帮助用户系统掌握 HTML、CSS 等前端基础知识。系统包含学习者移动端和管理后台 Web 端两部分。

---

## 3. 学习者端使用说明

### 3.1 账户管理

**注册**：打开 APP → 点击"注册" → 输入邮箱和密码（8-20位字母数字组合）→ 完成注册自动登录。  
**登录**：打开 APP → 输入已注册的邮箱和密码 → 点击"登录"。  
**编辑资料**：进入"我的" → 点击个人资料 → 修改昵称/头像/学习偏好 → 保存。

### 3.2 课程学习

**课程列表**：首页或"课程"Tab 查看所有已发布课程，显示完成进度。  
**章节学习**：点击课程进入详情 → 选择章节 → 阅读图文教学内容 → 点击"完成学习"标记完成。  
**学习进度**：首页显示继续学习入口，课程卡片显示百分比进度条。

### 3.3 编码练习

进入章节练习 → 阅读题目描述 → 在代码编辑器中编写代码 → 点击"运行"预览效果 → 点击"提交"进行评测。评测结果即时返回，显示得分和错误信息。可重复提交，系统记录最佳成绩。

### 3.4 AI 辅助

在练习页面点击"AI 帮助"按钮，可选择：  
- **错误解释**：分析代码错误原因并提供修复建议  
- **提示**：提供与当前问题相关的知识点提示  
每日 AI 调用有次数限制（默认 50 次/天）。

### 3.5 闯关挑战

进入"挑战"Tab → 查看关卡地图 → 选择已解锁关卡 → 完成关卡内的学习任务和练习 → 获得星级评定（1-3 星）。通关后获得经验值奖励，后续关卡自动解锁。

### 3.6 每日挑战

进入"每日挑战"页面 → 查看当日题目和限时倒计时 → 点击"开始挑战"作答 → 提交后即时获得结果。每日 0 点刷新，当日仅可参与一次。连续完成可获得额外奖励。

### 3.7 社交功能

**好友**：进入"社交"Tab → 搜索用户 → 发送好友请求 → 对方确认后建立好友关系。  
**排行榜**：查看全局、好友和课程排行榜，了解学习排名。  
**动态**：查看好友的学习动态（通关、获得徽章等）。

### 3.8 奖励与统计

**经验值**：完成章节、练习、挑战均可获得 XP，累积提升等级。  
**徽章**：达成特定条件自动解锁成就徽章，可在个人中心查看。  
**学习统计**：查看学习时长、完成关卡数、提交次数、连续天数等数据。

### 3.9 设置

在"我的"页面进入设置，可调整学习偏好（每日目标时长、难度偏好）、修改密码、管理登录设备。

## 4. 管理员使用说明

### 4.1 登录

在浏览器中打开管理后台 → 输入管理员账号和密码 → 进入仪表盘。

### 4.2 仪表盘

查看核心运营数据：用户总数、课程数、提交总数、活跃用户趋势等。数据每日凌晨 3 点自动更新。

### 4.3 课程管理

创建/编辑/上下架课程：设置课程名称、描述、难度、封面图。课程发布前必须包含至少一个章节。已下架课程新学员不可见，已选学员可继续学习。

### 4.4 章节管理

在课程内添加章节：设置标题、图文内容（Markdown 格式）、预计学习时长。章节顺序支持拖拽调整。

### 4.5 题目管理

创建练习题：选择类型（选择题/编程题），配置题目描述、选项和答案（选择题）或初始代码模板和测试用例（编程题）。编程题必须配置至少 1 个测试用例。

### 4.6 挑战管理

创建闯关挑战：设置名称、关联课程、关卡阶段、星级评定规则、奖励经验值。

### 4.7 用户管理

查看用户列表和详情、搜索用户、禁用/启用账号。禁用后用户无法登录，但历史数据保留。

### 4.8 反馈处理

查看用户提交的反馈工单 → 回复并处理 → 状态流转（open → in_progress → resolved → closed）。

### 4.9 内容审核

查看待审核的昵称和头像 → 通过/驳回处理。处理结果对用户可见。

### 4.10 公告管理

创建和发布公告：设置标题、正文、有效期、受众。仅与学习相关的公告内容。

### 4.11 系统配置

管理系统运行参数：AI 配置、积分规则、徽章配置等。关键操作需二次确认，审计日志保留 90 天。

## 5. 异常处理

| 常见问题 | 原因 | 解决方法 |
|---------|------|---------|
| 无法登录 | 账号密码错误或账号被禁用 | 检查密码是否正确，联系管理员确认账号状态 |
| 页面加载失败 | 网络异常或服务暂时不可用 | 检查网络连接，稍后重试 |
| 代码提交无响应 | 网络中断或服务繁忙 | 检查网络，等待后重新提交 |
| AI 帮助不可用 | 已达每日限额或服务离线 | 次日再试或稍后重试 |
| 每日挑战无法开始 | 今日无挑战配置 | 联系管理员配置每日挑战 |
